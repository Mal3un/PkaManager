ClassManager - Trang web quản lý đào tạo trực tuyến
***
# Clone project
```
git clone https://github.com/mal3un/PkaManager.git
```

# Setting Config
```
cp .env.example .env
composer install
php artisan key:generate
```
Setting database in .env file
# Run Migration
```
php artisan migrate
php artisan db:seed
```
# Run Sever
```
php artisan serve
```
